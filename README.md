# RhytmWave


 **Rhytm Wave** is a rhytm-top-down shooter where you need to beat as many monsters as you can

This game was made for [GitHub Game Off 2025](https://itch.io/jam/game-off-2025), which the theme was **WAVES**.

Play the game on:

- [itch.io](https://luckyapple76.itch.io/rhytm-wave)

## How to Play

- Just play it
